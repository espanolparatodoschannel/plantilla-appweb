# Prompt: Experto en Traducción de Contenido Pedagógico JSON

**Instrucciones para la IA:**
Actúa como un experto traductor de materiales educativos para la enseñanza del español (ELE) y desarrollador front-end. Tu tarea es traducir el archivo JSON de contenido de una página web pedagógica a 4 idiomas: **Inglés (en), Francés (fr), Portugués (pt) y Alemán (de)**.

### Reglas de Oro que DEBES seguir:
1. **INTEGRIDAD TÉCNICA (LLAVES):** NO traduzcas las llaves (keys) del JSON. Mantén exactamente igual: `meta`, `fichas`, `resumen`, `id`, `categoria`, `tema`, `explicacion`, `ejemplos`, `formato`, `items`, `pregunta`, `respuesta`, `vocabulario_util`, `consejo`, `titulo`, `intro`, `pasos`, `texto`, `ejemplo`, `color`, `ejemplo_completo`.
2. **VALORES TÉCNICOS:** NO traduzcas los valores de configuración. Mantén: `"lista"`, `"dialogo"`, `"grid"` y nombres de colores/clases de Tailwind como `"text-indigo-500"`, `"text-emerald-500"`, etc.
3. **ETIQUETAS HTML:** Mantén intactas todas las etiquetas HTML dentro de los textos (ej: `<strong>`, `<em>`, `<br>`, `<span>`, `class="..."`). Son necesarias para el diseño.
4. **DATOS FIJOS:** NO traduzcas nombres de archivos de imagen (en `ilustracion`) ni la `video_url`.
5. **ESTILO PEDAGÓGICO:** La traducción debe ser natural, clara y adecuada para un estudiante de nivel A1.
6. **GHOST TEXT (MAPEO):** La traducción debe mapear exactamente al texto original para que el efecto visual "Ghost Text" de la web sea coherente.
7. **TRADUCE LOS EJEMPLOS Y EL VOCABULARIO:** Como este JSON servirá para un efecto de "Ghost Text" (traducción superpuesta), DEBES traducir también las frases en español de los "ejemplos", el "vocabulario_util" y el "ejemplo_completo" al idioma destino. Por ejemplo, si el original dice "¡Hola, Elena!", la versión en inglés debe decir "Hello, Elena!". No dejes las frases de ejemplo en español, tradúcelas para que el alumno pueda ver su significado.

### Formato de Entrega:
Genera 4 bloques de código independientes. Cada bloque debe estar listo para ser guardado como un archivo individual con estos nombres:
- `contenido_en.json`
- `contenido_fr.json`
- `contenido_pt.json`
- `contenido_de.json`
